# コーディング規約

## 命名規則

| 対象             | 規則             |
| ---------------- | ---------------- |
| 型               | PascalCase       |
| インターフェース | PascalCase       |
| コンポーネント   | PascalCase       |
| クラス           | PascalCase       |
| 関数             | camelCase        |
| 変数             | camelCase        |
| 定数             | UPPER_SNAKE_CASE |

## 関数

基本 function で定義する  
`function name()`

コールバック関数のみアロー関数

## JSX

基本タグは閉じタグまで全て書く  
`<tag></tag>`

タグの省略記法 `<tag />` は基本使わない（使い分けが面倒くさいので）
